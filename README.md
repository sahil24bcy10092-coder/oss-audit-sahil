## Project Overview
This project is an open source audit of Git, a distributed version control system widely used in software development.

## Scripts Description

### Script 1: System Identity Report
Displays system information like kernel, user, uptime and date.

### Script 2: FOSS Package Inspector
Checks whether Git is installed and displays its version.

### Script 3: Disk and Permission Auditor
Analyzes important directories and their permissions.

### Script 4: Log File Analyzer
Searches logs for specific keywords and displays results.

### Script 5: Open Source Manifesto Generator
Generates a text file based on user input about open source philosophy.

## How to Run

Make scripts executable:
chmod +x script1.sh

Run script:
./script1.sh

For log analyzer:
./script4.sh /var/log/syslog
